# AfraMedico Operating System

## Changelog

This changelog records the official evolution history of the AfraMedico Operating System.

Every Sprint represents a meaningful step in the evolution of the platform. Architecture, documentation, product philosophy, frontend modules, and backend milestones are equally important because this repository records the growth of an operating system, not merely a sequence of code commits.

This file captures the history of the product at a high level. Detailed technical information remains inside `/docs`.
