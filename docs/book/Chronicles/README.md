# ICOS Chronicles

This folder records the living history of ICOS development.

These are not technical documents.

These are project chronicles.

Each chronicle should capture:

- Date.
- What changed.
- What was discovered.
- Why it mattered.
- What decision was made.
- What we learned.

The purpose is to preserve the evolution of the platform as it happens.

ICOS should not only preserve operational knowledge for future customers.

It should also preserve the history of its own creation.
