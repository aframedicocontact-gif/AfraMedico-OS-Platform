# ICOS Book Manuscript

This folder is intentionally different from documentation.

Documentation explains HOW.

The Book explains WHY.

These files are the living manuscript of the future ICOS book. They record the evolution of ideas, architecture decisions, business philosophy, operational discoveries, and design thinking behind the Intelligent Care Operating System.

This is not a user manual.

This is not API documentation.

This is the story of how operational knowledge became a software platform.

Future editions of the book should originate from these files rather than being written from memory.

## Working Rule

Never rewrite history.

Append ideas.

Preserve chronology.

The evolution of thinking is part of the value.
